■ 플러그인 설치방법
  ● 대시보드/플러그인/새로추가
  ● 검색설치 혹은 플러그인파일(zip파일)을 업로드
  ● 설치 후 활성화
  ● 설치된 플러그인 메뉴는 대시보드메뉴에, 설정, 도구메뉴에 등록된다

■ 플러그인 출력방법
  ● Shortcode를 관리자화면에서 추가, 코드편집기를 사용하길 권장함
    - 예시 : [code]
  ● template을 이용해 출력
    - 예시 : <?php echo do_shortcode( '[제공되는 Shortcode를 넣으면 된다.]' ); ?> 
■ Remove Category URL 
  ● url의 category출력을 제거
  ● 별도의 설정이 필요하지 않음

■ Breadcrumb NavXT
  ● 웹사이트 Breadcrumb 출력
  ● 대시보드/설정 안으로 메뉴가 등록됨
  ● 사용법
    -출력함수
    <?php if(function_exists('bcn_display')){ bcn_display(); }?>
  ● 세부옵션
    - General / Breadcrumb Separator : 구분기호
    - General / Home Template , Home Template (Unlinked) : 홈출력설정
    - 페이지나 글 출력방식 설정
      · Post Types / 글 template 안쪽의 내용은 여백 한칸 추가
      · Post Types / 글 template 또는 페이지 template 태그를 수정
      · Texonomies / 카테고리 template 태그를 수정

■ Regenerate Thumbnails
  ● 대표이미지 크기등을 일괄처리 재설정
  ● 사용법
    - 대시보드/설정/미디어/썸네일크기 변경후 저장
    - 대시보드/도구/Regenerate thumbnails
      · delete thumb... 체크, 
      · 버튼클릭

■ Contact form 7
  ● 관리자(대시보드 -> 설정 -> 일반 -> 관리자 이메일 주소로 보내짐)에게 mail 보내기!
  ● 사용법
    - 설치 후 대시보드에 연락처 메뉴(contact)가 추가됨
    - 제공되는 Shortcode를 해당 페이지에서 출력 
■ Kboard
  ● 게시판 plugin
  ● 대시보드 -> kboard 메뉴 생성됨
    - 게시판 목록 및 관리
      · 출력될 페이지 선택
      · 스킨(디자인)
      · 최근게시물 코드(아래 코드로 사용한다!)
          <?php 
            $homeURL = home_url(bloginfo('url'));
            $kboardID = "1" ;		
            $kboardPAGE = "notice";
            $kboardCnt = "3";
            $kboardSTR = '[kboard_latest id="1" url="'.$homeURL.$kboardPAGE.'" rpp="'.$kboardCnt.'"]'; 
            echo do_shortcode( $kboardSTR ); 
          ?>
    - 게시판 생성
      · 기본 설정
      · 권한 설정
        ▲ 읽기, 쓰기 권한을 설정한다.
        ▲ 제한없음, 로그인 사용자, 직접설정(관리자, 편집자, 글쓴이 등등 개별적으로 선택가능)
■ 위젯설정
  ● functions.php 
    register_sidebar( array(
      'name' => __( '위젯이름' ),
      'id'=> 'widjet1',  // 위젯 영역의 고유한 ID
    ));
  ● 대시보드/외모/위젯
    - 위젯이름으로 생성된 위젯 추가버튼 후 모두보기
    - 위젯 추가후 update
  ● 출력
    - 출력을 원하는 탬플릿에 하단의 코드 입력
    <?php
      if ( is_active_sidebar( 'widjet1' ) ) {
        dynamic_sidebar( 'widjet1' );
      }
    ?>
